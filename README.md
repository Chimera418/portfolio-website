# 👾 CHIMERA'S REALM
[![Live Demo](https://img.shields.io/badge/Live_Demo-chimera--realm.foo-4CAF50?style=for-the-badge)](https://chimera-realm.foo)
> An interactive portfolio filled with projects, experiments, and a few unnecessary distractions.

Welcome to my portfolio website! This is my small corner on the internet where I post my projects, random stuff, and all things code. 

Built as a Bento-grid style dashboard, it features multiple interactive components, real-time API integrations, and customizable retro aesthetics like CRT scanlines and pixelated fonts.

*Inspired by [jasoncameron.dev](https://jasoncameron.dev/)*
## 🛠️ Tech Stack

### 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Chimera418/portfolio-website.git

# Install dependencies
npm install

# Start the dev server
npm run dev
```
